# Neon Survivors Prototype

이 저장소는 GitHub Pages에서 바로 실행 가능한 탑다운 생존 게임 프로토타입을 제공합니다. [index.html](./index.html) 한 파일에 HTML, CSS, JavaScript가 포함되어 있으며, 터미널/Bash 스크린세이버 감성을 반영한 네온 스타일을 사용했습니다.

## 게임 특징

- **자동 전투 + 성장**: WASD/화살표 키로 이동하면 가장 가까운 적을 자동으로 조준/발사합니다. 적이 떨어뜨리는 데이터 조각을 모아 경험치를 획득하고 레벨을 올리면 화력, 체력, 발사 속도가 강화됩니다.
- **다채로운 적 유형**: 글리치 비트, 데몬 위습, 커널 브루저, 토렌트 브루드 등 움직임과 능력치가 서로 다른 적 무리가 시간 경과에 따라 출현합니다.
- **루팅 시스템**: 적 처치 시 데이터 조각(XP), 에너지 셀(체력 회복), 오버클럭 코어(임시 발사 속도 증가), 페이즈 쉴드(방어 강화) 등 다양한 드롭을 줍습니다.
- **업데이트된 HUD**: 생존 시간, 처치 수, 체력, 레벨, 경험치 게이지, 활성화된 버프를 실시간으로 확인할 수 있습니다.
- **웹 전용 프로토타입**: 별도의 빌드 과정 없이 최신 브라우저에서 `index.html`을 열기만 하면 됩니다.

## 파일 구성

- `index.html`: 게임 렌더링 캔버스, 업데이트 루프, 적/아이템/총알/플레이어 로직, UI 스타일을 모두 포함합니다.
- `_config.yml`: GitHub Pages 기본 설정 파일로, 필요 시 테마나 옵션을 추가할 수 있습니다.
- `favicon.ico`: 사이트 파비콘 자리입니다.

## 실행 방법

1. 이 저장소를 GitHub Pages로 배포하거나 `index.html`을 로컬 브라우저에서 직접 엽니다.
2. 페이지가 로드되면 즉시 시뮬레이션이 시작되며, WASD 또는 화살표 키로 이동합니다.
3. 적을 처치하고 떨어지는 전리품을 모아 레벨을 올리세요.
4. 버프를 활용해 가능한 오래 생존 시간을 늘려 보세요. 게임 종료 후 Enter 키나 화면 클릭으로 재부팅할 수 있습니다.

> ⚠️ 이 버전은 사운드, 업그레이드 선택 UI, 모바일 조작이 없는 프로토타입입니다. 후속 작업에서 확장할 수 있습니다.
