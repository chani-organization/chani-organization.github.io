# Neon Survivors Prototype

이 저장소는 GitHub Pages에서 바로 실행 가능한 탑다운 생존 게임 프로토타입을 제공합니다. [index.html](./index.html) 한 파일에 HTML, CSS, JavaScript가 포함되어 있으며, 터미널/Bash 스크린세이버 감성을 반영한 네온 스타일을 사용했습니다.

## 게임 특징

- **자동 전투**: 플레이어는 WASD/화살표 키로 이동만 담당하며, 가장 가까운 적을 0.75초마다 자동 조준 및 발사합니다.
- **끝없는 웨이브**: 적이 사방에서 등장해 플레이어에게 돌진합니다. 시간이 지날수록 등장 주기가 짧아지고 이동 속도가 빨라집니다.
- **간결한 HUD**: 생존 시간, 처치 수, 체력 상태를 실시간으로 표시합니다. 체력이 0이 되면 게임이 종료되고 Enter 또는 클릭으로 재시작할 수 있습니다.
- **웹 전용 프로토타입**: 별도의 빌드 과정 없이 최신 브라우저에서 `index.html`을 열기만 하면 됩니다.

## 파일 구성

- `index.html`: 게임 렌더링 캔버스, 업데이트 루프, 적/총알/플레이어 로직, UI 스타일을 모두 포함합니다.
- `_config.yml`: GitHub Pages 기본 설정 파일로, 필요 시 테마나 옵션을 추가할 수 있습니다.
- `favicon.ico`: 사이트 파비콘 자리입니다.

## 실행 방법

1. 이 저장소를 GitHub Pages로 배포하거나 `index.html`을 로컬 브라우저에서 직접 엽니다.
2. 페이지가 로드되면 즉시 시뮬레이션이 시작되며, WASD 또는 화살표 키로 이동합니다.
3. 적과 부딪히지 않도록 회피하면서 가능한 오래 버티세요. 게임 종료 후 Enter 키나 화면 클릭으로 재부팅할 수 있습니다.

> ⚠️ 이 버전은 기본 조작과 난이도 곡선만 갖춘 프로토타입입니다. 후속 작업에서 무기/레벨업 시스템, 업그레이드, 사운드, 모바일 조작 등을 확장할 수 있습니다.
